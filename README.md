# DecodeLabs_Project.3_Ayomide_Adesola_Akande
# 🗄️ SQL Analysis of E-Commerce Sales Data

## 📌 Overview
This project involved analyzing an e-commerce sales dataset using Microsoft SQL Server to extract meaningful business insights. SQL queries were used to examine sales performance, customer behavior, payment preferences, referral sources, and operational trends.

## 🎯 Objectives
- Identify products generating the highest revenue.
- Determine high-value customers.
- Analyze customer payment preferences.
- Evaluate order status distribution.
- Examine revenue trends over time.
- Identify the highest-performing referral source.

## 🛠️ Tools Used
- Microsoft SQL Server
- SQL
- Data Analysis
- Business Intelligence

## 📈 Key Metrics
- **Total Orders:** 1,200
- **Total Revenue:** ₦1,264,761.96
- **Average Order Value:** ₦1,053.97

## 🔍 Key Insights
-  Chairs generated the highest revenue.
-  Top customers contributed significantly to total sales.
-  Online Payment was the most preferred payment method.
-  41.4% of orders were cancelled or returned.
-  Revenue peaked in June.
-  Instagram generated the highest referral revenue.

## 💡 Recommendations
-  Prioritize high-performing products.
-  Strengthen customer retention efforts.
-  Maintain secure online payment systems.
-  Investigate causes of cancellations and returns.
-  Replicate strategies from peak revenue periods.
-  Increase marketing efforts on Instagram.

## 🏁 Conclusion
The analysis transformed raw transactional data into actionable business insights. The findings highlight opportunities to improve customer retention, optimize marketing strategies, and enhance overall business performance.
